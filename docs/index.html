Model Compare Tetris

Each card opens a different Tetris generation built from the same prompt by a different AI model. Every demo is published in its own subdirectory so the deployed URL stays aligned with the folder name in this repository.

The prompts were run in planning mode, and when the agent asked follow-up multiple-choice questions it generally took the first or recommended option.

After that, each run was allowed to continue autonomously until completion.

prompt.md
> Create a tetris clone with an immersive retro aesthetic. > The game starts automatically at the main menu. > Main menu entries are for new game options, settings, and about.
