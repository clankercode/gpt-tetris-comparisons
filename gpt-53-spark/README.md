# Neon Block Cascade

Retro-styled Tetris clone built with **Bun + TypeScript + Vite + Phaser**.

## Quick Start

- Install dependencies:
  - `bun install`
- Start the game with a Python-randomized dev port:
  - `bun run dev`
- Or run a fixed-port Vite preview:
  - `bun run preview`
- Build:
  - `bun run build`
